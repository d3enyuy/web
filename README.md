# Portfolio

Personal portfolio site built with Next.js, TypeScript, and Tailwind CSS.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

## Dev

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Live at [lambivgills.com](https://lambivgills.com), deployed on Cloudflare Pages.
